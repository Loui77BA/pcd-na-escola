      ### Boas-vindas e Contexto

      O Windows 10 começou a entrar em processo de aposentadoria, por conta disso é necessário atualizar o curso de Windows para pessoas com baixa visão dessa vez para o Windows 11.

      ### Apresentação

      Olá mundo! Quem vai conduzir esse curso é a Louise Suelen, que é uma pessoa com deficiência visual, cega, é engenheira de software e sobrevivente do bacharelado em ciência da computação.

      ### Recados Importantes

      O processo inteiro será realizado com o uso de um leitor de telas, então o processo estará sendo narrado, porém é importante que outras pessoas cegas saibam que sons específicos foram usados para cada tipo de recurso visual que foi utilizado.

      Por exemplo, todas as vezes que uma palavra for destacada, um som específico será emitido (o mesmo som todas as vezes que uma palavra for destacada), o mesmo acontece com as Transições no vídeo, um som específico é o mesmo para todas as transições.

      Apresentações feitas e recados dados, vamos começar o curso.
